# 今日数学知识点速记版

> 主题：函数性质、取整函数、周期函数、原函数、变限积分  
> 适用：考研数学选择题、填空题、概念辨析题

---

## 1. 常数函数的性质

设：

```math
f(x)=C
```

### 速记结论

常数函数：

- 是偶函数；
- 是周期函数；
- 可以认为任意正数都是它的周期。

### 判断理由

偶函数要求：

```math
f(-x)=f(x)
```

常数函数中：

```math
f(-x)=C,\quad f(x)=C
```

所以：

```math
f(-x)=f(x)
```

周期函数要求存在 `T > 0`，使得：

```math
f(x+T)=f(x)
```

常数函数中：

```math
f(x+T)=C,\quad f(x)=C
```

所以任意 `T > 0` 都满足。

### 易错点

不要以为“周期”一定有最小正周期。

常数函数虽然是周期函数，但没有最小正周期。

## 2. 取整函数的奇偶性

取整函数：

```math
[x]
```

表示不超过 `x` 的最大整数，也叫向下取整。

例如：

```math
[1.3]=1,\quad [-1.3]=-2
```

### 速记结论

`[x]` 既不是奇函数，也不是偶函数。

### 关键关系

当 `x` 不是整数时：

```math
[-x]=-[x]-1
```

当 `x` 是整数时：

```math
[-x]=-[x]
```

### 秒判例子

取 `x = 1.3`：

```math
[1.3]=1
```

```math
[-1.3]=-2
```

而：

```math
-[1.3]=-1
```

所以：

```math
[-x]\neq -[x]
```

因此不是奇函数。

又因为：

```math
[-1.3]\neq [1.3]
```

所以也不是偶函数。

### 易错点

取整函数不能简单认为：

```math
[-x]=-[x]
```

这个式子只有在 `x` 为整数时成立。

## 3. 变限积分求导公式

### 3.1 上限是 `x`

若：

```math
F(x)=\int_a^x f(t)\,dt
```

则：

```math
F'(x)=f(x)
```

### 3.2 下限是 `x`

若：

```math
F(x)=\int_x^a f(t)\,dt
```

则：

```math
F'(x)=-f(x)
```

### 3.3 上限是复合函数

若：

```math
F(x)=\int_a^{\varphi(x)} f(t)\,dt
```

则：

```math
F'(x)=f(\varphi(x))\varphi'(x)
```

### 3.4 上下限都是函数

若：

```math
F(x)=\int_{\alpha(x)}^{\beta(x)} f(t)\,dt
```

则：

```math
F'(x)=f(\beta(x))\beta'(x)-f(\alpha(x))\alpha'(x)
```

### 速记口诀

> 上限代入乘上限导，  
> 下限代入乘下限导，  
> 下限整体前面加负号。

## 4. 积分上限变成 `-x` 怎么处理？

常见形式：

```math
\int_0^{-x} f(t)\,dt
```

### 不能直接写成

```math
-\int_0^x f(t)\,dt
```

是否能这样变，要看 `f(t)` 的奇偶性。

### 通用变形

令 `t = -u`，有：

```math
\int_0^{-x} f(t)\,dt
=
-\int_0^x f(-u)\,du
```

也就是：

```math
\int_0^{-x} f(t)\,dt
=
-\int_0^x f(-t)\,dt
```

### 若 `f` 是偶函数

偶函数满足：

```math
f(-t)=f(t)
```

所以：

```math
\int_0^{-x} f(t)\,dt
=
-\int_0^x f(t)\,dt
```

### 若 `f` 是奇函数

奇函数满足：

```math
f(-t)=-f(t)
```

所以：

```math
\int_0^{-x} f(t)\,dt
=
\int_0^x f(t)\,dt
```

### 易错点

符号能不能提出去，不只看积分上限，还要看被积函数的奇偶性。

## 5. 原函数与连续性

若：

```math
F'(x)=g(x)
```

这表示 `F(x)` 是 `g(x)` 的原函数。

若题目还告诉 `g(x)` 连续，则说明 `F(x)` 可导，且导函数连续，即：

```math
F(x)\in C^1
```

### 速记结论

“`F` 是 `g` 的原函数”就是告诉你：

```math
F'=g
```

“`g` 连续”就是告诉你：`F'` 连续。

### 常见用途

用于判断：

- 函数是否可导；
- 分段点处参数如何确定；
- 能否使用变限积分求导；
- 能否使用牛顿-莱布尼茨公式。

---

## 6. 由原函数关系求参数

如果题目说 `F(x)` 是 `g(x)` 的原函数，不能直接推出某个参数。

正确做法：

1. 对 `F(x)` 求导；
2. 写出 `F'(x)`；
3. 令 `F'(x)=g(x)`；
4. 比较两边表达式；
5. 解参数。

### 速记

> 原函数关系的本质是求导后相等。

即：

```math
F'=g
```

## 7. 周期函数的原函数是否周期？

### 速记结论

周期函数的原函数不一定是周期函数。

设 `f(x)` 的周期为 `T`。

如果：

```math
F(x)=\int_0^x f(t)\,dt
```

要使 `F(x)` 也具有周期性，需要：

```math
\int_0^T f(t)\,dt=0
```

### 关键条件

```math
\boxed{\int_0^T f(t)\,dt=0}
```

即：一个周期内的面积为 0。

---

### 反例

令：

```math
f(x)=1
```

`f(x)` 是周期函数，因为常数函数任意正数都是周期。

但它的原函数：

```math
F(x)=x+C
```

不是周期函数。

原因是：

```math
\int_0^T 1\,dt=T\neq 0
```

---

### 正例

令：

```math
f(x)=\cos x
```

周期为：

```math
T=2\pi
```

其原函数可以为：

```math
F(x)=\sin x
```

也是周期函数。

因为：

```math
\int_0^{2\pi}\cos x\,dx=0
```

## 8. 奇偶性与原函数的关系

设：

```math
F(x)=\int_0^x f(t)\,dt
```

### 8.1 若 `f(x)` 是奇函数

则 `F(x)` 是偶函数，即：

```math
f(x)\text{ odd } \Rightarrow \int_0^x f(t)\,dt\text{ even}
```

### 例子

```math
f(x)=x
```

是奇函数。

则：

```math
F(x)=\int_0^x t\,dt=\frac{x^2}{2}
```

是偶函数。

---

### 8.2 若 `f(x)` 是偶函数

则 `F(x)` 是奇函数，即：

```math
f(x)\text{ even } \Rightarrow \int_0^x f(t)\,dt\text{ odd}
```

### 例子

```math
f(x)=\cos x
```

是偶函数。

则：

```math
F(x)=\int_0^x \cos t\,dt=\sin x
```

是奇函数。

---

### 易错点

这里说的是特殊原函数：

```math
F(x)=\int_0^x f(t)\,dt
```

不是所有原函数。

因为原函数可以加常数：

```math
F(x)+C
```

加上常数后，奇偶性可能改变。

例如：

```math
\sin x
```

是奇函数，但：

```math
\sin x+1
```

既不是奇函数，也不是偶函数。

## 11. 最后速背版

1. 常数函数：偶函数，周期函数，任意正数为周期。
2. 取整函数：一般既非奇也非偶。
3. 非整数时：`[-x]=-[x]-1`。
4. 变限积分求导：上限正，下限负，复合乘导。
5. `\int_0^{-x} f(t)\,dt` 不能乱提负号，要看 `f` 的奇偶性。
6. 原函数关系就是：`F'=f`。
7. 周期函数的原函数不一定周期。
8. 原函数要周期，必须一个周期内积分为 0。
9. 奇函数从 0 到 `x` 的积分是偶函数。
10. 偶函数从 0 到 `x` 的积分是奇函数。
